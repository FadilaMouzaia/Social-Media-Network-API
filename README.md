# Node-Social-Media-API
This project provides a RESTful API for social media application for managing post, profile, likes, and followings.
## Features
 - Signup and Signin
 - Follow/Unfollow users
 - update user
 - delete user
 - get a user
 - get friends
 - create a post
 - get a post
 - get timeline posts
 - get user's all posts
 - Like/Dislike Post
 - Update Posts
 - Delete Posts
